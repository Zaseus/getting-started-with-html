What is the Web?

The web is comprised of three main things. The Browser, Internet, and Servers.In essence, the web is a lot of computers talking to one another. A person sends an HTTP Request to a server which then returns the requested information. HTTP (which stands for Hyper Text Transfer Protocol) is comparable to the highways that connect cities to one another.The Browser is the indvidual machine that connects to the network or the Internet and sends requests to the Server


HTML?

HTML stands for Hyper Text Markup Language. It is what gives form and function to web pages. it's made up of two things, Text content and Markup. Text Contet makes up what you see in a webpage (content) while Markup decides how that content looks.


Elements and Tags

An Element refers to that which is everything between an opening and closing Tag. There are two types of Elements, Inline and Block. Inline is an Element that just makes a line. An example wold be < br >, < br > is a void tag that moves to the next line, or sometimes referred to as a Break. A Block Element creates an invisible Box that has specific height and width restrictions. < p > is a block Element called paragraph. A Tag is always contained within angled brackets. Most tags have an opening tag (< p >) followed by a closing tag (< /p >). There are some called Void Tags that do not require closing tags.

Some of the more commonly used tags are as follows:
< b > < /b > - The Bold tag
< em > < /em > - The Italics tag
< a > < /a > - This is the anchor tag.
The anchor tag is used to place Hyper Links or Links in the content of a webpage. < a href="www.reddit.com"> derp< /a > is an example of what can be used to insert a link in a page. It shows up as derp. (href) is the atrribute that is being assigned to the anchor.
